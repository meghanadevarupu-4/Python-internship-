# To-Do List CLI Application

## Description
A simple command-line To-Do List application built using Python. It allows users to add, view, complete, and delete tasks.

## Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- User-friendly menu

## Requirements
- Python 3.x

## How to Run

1. Open a terminal.
2. Navigate to the project folder.
3. Run the program:

```bash
python todo.py
```
